## musanzi-wilfried.me


### Clone the repo

```bash
git clone https://github.com/musanziw/portfolio--v1.git
```

### Install dependencies

```bash
pnpm i
```

### Run dev server

```bash
pnpm dev
```

### Build for production
```bash
npm run build
# or
yarn build
# or
pnpm build
```
